# User Features Proposal

## Taklif qilingan funksiyalar:

### 1. **Saved Projects (Saqlangan Project'lar)**
- User'lar o'z kodlarini server'da saqlashlari mumkin
- Project nomi, tavsifi, kategoriya qo'shish
- Project'larni ochish, tahrirlash, o'chirish
- Auto-save funksiyasi
- Project'larni qidirish va filterlash

### 2. **Personal Snippets Library (Shaxsiy Snippet'lar)**
- User'lar o'z snippet'larini yaratish va saqlash
- Snippet nomi, tavsifi, kategoriya
- Snippet'larni tez import qilish
- Favorites snippet'lar

### 3. **User Profile Modal**
- Profile settings
- Password o'zgartirish
- Phone number yangilash
- Account statistics

### 4. **Project Folders/Categories**
- Project'larni kategoriyalarga bo'lish
- Folder'lar yaratish va boshqarish
- Drag & drop bilan project'larni ko'chirish

### 5. **Projects Management Modal**
- Barcha project'larni ko'rish
- Project'larni qidirish
- Project'larni filterlash (tarix, kategoriya)
- Project'larni export qilish

### 6. **Cloud Sync**
- Avtomatik saqlash
- Boshqa device'larda ochish
- Real-time sync

### 7. **Export History**
- Export qilingan fayllar tarixi
- Qayta download qilish

### 8. **Code Comments/Notes**
- Project'larga izohlar qo'shish
- Kod qismlariga comment'lar
