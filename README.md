## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Usage

1. Select a tab (HTML, CSS, or JavaScript) to start editing
2. Type your code - changes are automatically saved
3. View the live preview on the right (desktop) or bottom (mobile)
4. Use the language toggle to switch between Uzbek, Russian, and English
5. Click "Run" to manually refresh the preview, or "Clear" to reset all code

## Browser Support

Works in Chrome, Safari, Firefox on Android, iOS, and desktop browsers.
