## Eng muhim funksiyalar (darhol qo'shish mumkin)

### 1. **Export/Download Funksiyasi** ⭐⭐⭐
- HTML, CSS, JS fayllarni alohida yuklab olish

- Export as ZIP - barcha fayllarni ZIP arxivida

### 2. **Dark Mode (Theme Switcher)** ⭐⭐⭐

### 3. **Code Formatting** ⭐⭐
- Prettier yoki o'z formatter

### 4. **Share via URL** ⭐⭐

### 5. **Font Size Control** ⭐⭐

### 6. **Fullscreen Mode** ⭐

- Mobile uchun ham ishlaydi
